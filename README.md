# Maven Market Report
# Project Description
Maven Market is a Power BI business intelligence project using a dataset from a fictional multinational grocery chain (USA, Canada, Mexico) to clean and model CSV data, then build interactive visuals and KPIs showing sales, profits, returns, and trends
# Technology Used
1. Power BI
2. Power Query
3. Dax Function
